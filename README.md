# Rampant2
Second Repo Rampant Game

Controls:
Escape to view stats customization page
F to view weapon select/drop menu
WASD to move
Left Shift to dash
Left Click to pull sword in and out
Right click(Hold) to hold sword horizontally
E to pick up weapons
Q to use potions
Mouse to aim
